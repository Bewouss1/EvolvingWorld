# Mise à jour 10/03 — Session EvolvingWorld

## Nouveaux fichiers

### Interface
- `Assets/Scripts/IInteractable.cs` — Interface partagée pour tous les interactables. Méthodes : `InteractionDistance` (float), `Interact()`. Permet d'ajouter de nouveaux types d'interactables sans toucher au PlayerController.

### Système Marchand
- `Assets/Scripts/Merchant/MerchantData.cs` — ScriptableObject (Create > Merchant > MerchantData). Contient `forSale[]` (MerchantOffer : item + stock) et `wantsToBuy[]` (ItemData[]). Ne pas modifier à runtime, le stock est copié dans NPCMerchant.
- `Assets/Scripts/Merchant/NPCMerchant.cs` — Composant NPC marchand. Implémente IInteractable. Gère le stock runtime en `int[]` (copie du SO au Awake). Méthodes publiques : `GetStock(index)`, `DecrementStock(index)`.
- `Assets/Scripts/Merchant/MerchantUI.cs` — Singleton UI marchand. Deux onglets Acheter/Vendre. Fermeture via bouton ou Échap. Champs Inspector : merchantPanel, slotsContainer, merchantSlotPrefab, buyTabButton, sellTabButton, closeButton, goldSprite, goldText, playerInventory.
- `Assets/Scripts/Merchant/MerchantSlotUI.cs` — Ligne de trade. Layout : [icône gauche + label] → [icône droite + label] + bouton action. Champs Inspector : leftIcon, leftLabel, rightIcon, rightLabel, actionButton, canvasGroup.

## Fichiers modifiés

- `Assets/Scripts/NPC/NPCInteractable.cs` — Implémente maintenant `IInteractable`.
- `Assets/Scripts/Player/PlayerController.cs` — Utilise `IInteractable` au lieu de `NPCInteractable`. Bloque ClickToMove si `MerchantUI.IsOpen`.

## Setup Unity requis

### MerchantUI GameObject (scène)
- Ajouter un GameObject avec le composant `MerchantUI`
- Créer le panel avec : TabBar (BuyTabButton + SellTabButton), SlotsContainer, CloseButton, GoldText
- Assigner tous les champs dans l'Inspector
- Assigner le `Inventory` du joueur

### MerchantSlotPrefab
- Root : Image de fond + Button (actionButton) + CanvasGroup + MerchantSlotUI
- Enfants : LeftIcon (Image), LeftLabel (TMP), flèche, RightIcon (Image), RightLabel (TMP)
- Raycast Target : décoché sur toutes les Images sauf le bouton action
- CanvasGroup : valeurs par défaut (le script gère alpha et blocksRaycasts)

### NPC Marchand
- Ajouter `NPCMerchant` sur le GameObject NPC (même layer interactable que NPCInteractable)
- Créer un `MerchantData` (SO) et l'assigner

## Règles importantes
- Le stock marchand est **runtime** (int[] dans NPCMerchant), le ScriptableObject MerchantData n'est jamais modifié.
- Pour griser un slot non disponible : `canvasGroup.blocksRaycasts = false` + `canvasGroup.alpha = 0.5`. Ne pas utiliser `interactable = false` seul (laisse passer les hover events et cause des effets visuels parasites).
- `inventoryPanel` doit être **désactivé dans l'éditeur** (pas seulement par code dans Awake), sinon `IsOpen` retourne true au démarrage et bloque le mouvement du joueur.
